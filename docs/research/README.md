# research/ — the thinking behind the canon

Evidence, analysis, and positioning that drive what the canon changes (and what
it doesn't). Not normative — the canon is. Dated where it matters; decide from
runs, not theory.

| File | What |
|---|---|
| `findings.md` | Findings from real runs and admissible external research, each with a candidate change. The decision log feeding amendments. |
| `specline-thesis.md` | The positioning and rationale — why Specline exists. |
| `fable-5-vs-specline.md` | Prior-art comparison against a frontier long-horizon model. |

**What's not here:** the rules themselves (root `specline-2.6.md`), or shaped
amendments ready to ratify (`docs/proposals/`).
