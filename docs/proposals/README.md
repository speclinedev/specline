# proposals/ — canon amendment shaping records

One record per amendment cluster. A proposal is shaped here, verified, ratified
by the decider, then **folded into the canon** and marked SUPERSEDED — kept for
provenance, never deleted. Where a proposal and the canon disagree, the canon
wins.
