# conventions/ — templates and standards a repo adopts

Reusable starting points for a Specline-governed repo. These are conventions, not
canon — a repo copies and adapts them.

| File | What |
|---|---|
| `specline-CLAUDE.md` | Template `CLAUDE.md` that teaches an agent to use Specline in a target repo. |

**What's not here:** the rules these conventions implement (root `specline-2.7.md`).
